Focus Buddy - Resume Project Description:
  Focus Buddy is a web-based study companion application developed using HTML, CSS, and JavaScript.
  The application helps students stay focused by providing structured study sessions, break reminders, motivational messages, and timer-based workflow management.

Key Features:
* Study Session Timer
* Break Timer
* Motivational Messages
* Multi-Screen Navigation
* Responsive User Interface

Technologies Used:
* HTML
* CSS
* JavaScript
* GitHub

Future enhancements include ESP32 hardware integration, OLED display support, buzzer notifications, and LED indicators.
