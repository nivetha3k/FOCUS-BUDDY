Focus Buddy

Project Description:
Focus Buddy is a student productivity application designed to help users stay focused during study sessions.

The application provides:
* Home Screen
* Study Session Screen
* Break Screen
* Session Complete Screen

Focus Buddy motivates students with encouraging messages and a simple workflow that helps improve concentration and productivity.

Features:
* Start Study Session
* Take Break
* Complete Session
* Motivational Messages
* Simple User Interface

Technologies Used:
* HTML
* CSS
* JavaScript
* GitHub

Future Enhancements:
* Real Timer
* Focus Score Calculation
* Study Streak Tracking
* Focus Buddy Robot Assistant
* ESP32 Hardware Integration
