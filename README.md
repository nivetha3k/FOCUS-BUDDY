FOCUS BUDDY

Problem Statement: 
         Students often spend hours at their desks without knowing how productive or focused they actually are.

Solution:
      Focus Buddy is a smart study companion designed to help students track study sessions, monitor breaks, and calculate a focus score to improve productivity.

Features:
* Study session timer
* Break tracking
* Focus score calculation
* Productivity dashboard
* Future IoT integration with ESP32

Technologies:
* HTML
* CSS
* JavaScript
* GitHub
* Future: ESP32 and IoT
